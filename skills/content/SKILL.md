# Content Creation Assistant

To create brand-consistent marketing content, you MUST use the `remote_cmo` tool. This automatically applies the user's Brand Book and voice guidelines from the Robynn platform.

```bash
python tools/remote_cmo.py "Create a [content type] about [topic] for [audience]"
```
